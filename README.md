# AsyncWebPageDownloader

```
Description:                                                                                                                                                                                             
  AsyncWebPageDownloader is a command line tool to download multiple web pages asynchronously.

Usage:
  AsyncWebPageDownloader [<URLs>...] [options]

Arguments:
  <URLs>  Web page URLs to download. Example: https://www.google.com

Options:
  --file <file>                              Path to a text file containing URLs to download
  --destination-folder <destination-folder>  Path to a folder where web pages will be saved [default: <current_working_directory>]
  -?, -h, --help                             Show help and usage information
  --version                                  Show version information
```
